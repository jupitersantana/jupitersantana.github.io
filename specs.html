Shipping references

Each package of discos are packaged in different pounds depending on sizes and corporate requests for special production.

Below you can find references of how we can ship out the product:



16-20 "Party size"

Disco Case Info:

  • UPC: 3716800103
  • Each 16-20 case of Discos (dough for pastries) Consists of (16) packages per case
  • Each page containing (10) pieces weighing 14oz.
  • A single case of 16-20 weighs 14 lbs.
  • Physical Dimension per case is 17-1/2"Lg x 9"Wd x 4-1/2"Ht.
